Scraping Start-Up Drama Ratings and Comments using Python

Note: Only for educational purpose.

by Michairyniza Richlellyn and Lim Suying (2020).


Introduction

The ratings data were scraped from the official Internet Movie Database (IMDb) website. IMDb is one of the world’s most popular sources for movie, TV and celebrity content with more than 100 million unique visitors per month. IMDb has huge collection of movies database that includes various details of movies along with different ratings and user reviews. This movie reviews affects everyone from audience, film critics to the production company.

Meanwhile, the reviews data were scraped from the Reddit site. Reddit is a social news platform that allows users to discuss and vote on content that other users have submitted. To help police the site and prevent spammers from bombarding readers, Reddit came up with “karma” points. Users get karma by their comments and links being up-voted by others in the community.


Objectives

The idea of our project is to scrape the data from IMDb and Reddit, and form an analysis that will help users or viewers to view the ratings and comments for this specific Korean drama, Start-Up.

  1. Extract and visualise the rating of each episode.
  2. Extract and visualise the overall user ratings.
  3. Extract and visualise the rating by demographic.
  4. Extract and visualise the comments for each episode by using WordCloud.
  5. Extract and visualise the frequently used words in the comments using tri-gram.

Tools and Methods
  1. Jupyter Notebook
    • User Ratings
      • Import all necessary libraries: BeautifulSoup, Seleniu, ChromeDriver, WebDriver, Pandas, Matplotlib and Plotly
      • Get url: https://www.imdb.com/title/tt12867810/
      • Parse HTMl by using BeautifulSoup.
      • Extract data wanted: user ratings.
      • Save extracted data to DataFrame.
      • Remove unwanted characters (non-numerical, string, new lines etc).
      • Save to csv file.
      • Visualise data by using Plotly/Matplotlib.

    • Reddit Comments
      • Import all necessary libraries: csv, request, Pandas, Numpy, Matplotlib, re, nltk (stopwords, wordnet, FreqDist, PorterStemmer, RegexpTokenizer, sent_tokenize, word_tokenize, WordNetLemmatizer), praw, BeautifulSoup, WordCloud, gensim, Collections, sklearn and SpellChecker.
      • Praw: Reddit Web Scraping
      • Data scrapping and storing
        1. Query with keyword 'Start-up'
        2. Create dictionary for posts and comments
        3. Scrap data for top 20 posts sorted by comments
        4. Store comments and posts scrapped into csv file
      • Data cleaning: Lookup post title to identify episode number
      • Replacing words, removing emojis and URLs
      • Remove emoji, URLs, stopwords, punctuations
      • Remove deleted comments
      • Data Visualisation: WordCloud & Tri-gram
      • Save DataFrame to csv
